Синтезатор звука на заданных частотах

Использование SoundSynthesizer [file in] [file out]

Программа получает на вход файл формата:
	1.Первая строка - частота частота дискретизации в герцах (целое число)
	2.Последующие строки - частота звука в герцах, через пробел длительность в секундах (дробные числа)

На выходе программа сохраняет аудиофайл в формате .wav
